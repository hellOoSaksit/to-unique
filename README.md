# ถึงยูนิก ✨ — จดหมายจากดวงดาว

หน้าเว็บบอกรักแบบอินเตอร์แอคทีฟ: กดซองจดหมายจากดาว → ดาวกระจายขึ้นฟ้า → ซูมเข้าไปอ่านจดหมายหลายหน้า พร้อมเนื้อเพลงวิ่งซ้าย/ขวา ดาวตก และหัวใจดาวเรืองแสง

เว็บทั้งหมดอยู่ในไฟล์เดียว [`index.html`](index.html) (HTML + CSS + JS) เปิดในเบราว์เซอร์ได้เลย ไม่ต้องติดตั้งอะไร

## ดูออนไลน์

เผยแพร่ผ่าน GitHub Pages — ดูลิงก์ได้ที่ส่วน **About / Deployments** ของ repo นี้

## ใส่รูปและเพลงของคุณเอง

โค้ดอ้างอิงไฟล์ในโฟลเดอร์ `assets/` — ถ้าไม่มีไฟล์ หน้าเว็บยังทำงานได้ (รูปจะเป็นลายพื้นหลัง, ไม่มีเสียง) แต่ถ้าอยากได้ครบให้วางไฟล์เหล่านี้:

| ไฟล์ | ใช้ที่ไหน |
|------|----------|
| `assets/pic1.jpg` | รูปหน้าจดหมายหน้าที่ 1 |
| `assets/pic2.jpg` | รูปหน้าจดหมายหน้าที่ 2 |
| `assets/pic4.jpg` | รูปหน้าจดหมายหน้าที่ 3 |
| `assets/stamp.png` | สแตมป์มุมขวาบนของจดหมาย + หน้าซอง |
| `assets/song.mp3`  | เพลงประกอบ (ยาว ~1:40) |

> ⚠️ **เรื่องลิขสิทธิ์เพลง:** โปรเจกต์ต้นทางตั้งใจใช้เพลง *Ed Sheeran – Perfect* ซึ่งมีลิขสิทธิ์ ไม่ได้ถูกอัปโหลดขึ้นเว็บสาธารณะนี้ ถ้าจะเผยแพร่ต่อสาธารณะควรใช้เพลงที่คุณมีสิทธิ์ใช้งาน
